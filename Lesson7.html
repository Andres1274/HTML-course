Developers group eleents to create unique
layouts for their websites like the one below



______________________________________

The purple boxes highlight how this webpage
groups elements into a navigation menu, a
top story, other stories, and the lastest news.

______________________________________

To create a group of elements , we surround
the elements with < div > and < / div >
tags.
index.html
_____________________________________________________

< ! doctype html >
< html >
< body >
< div >
< h3>lastest News< / h3 >
< h1>Where VC's are investing in
Fintech< / h1 >
< p>Arman< / p>
< / div >
< / body >
< / html >


______________________________________

The < div >< / div > tags and the elements
between them make up one group on a
webpage, also known as a div element.
index.html
_____________________________________________________

< ! doctype html >
< html >
< body >
< div >
< h3>lastest News< / h3 >
< h1>Where VC's are investing in
Fintech< / h1 >
< p>Arman< / p>
< / div >
< / body >
< / html >


______________________________________

nested elements, for example, nesting
< h1 >Top Story< / h1 > inside < div >< / div >
